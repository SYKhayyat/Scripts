* System Package Managers

** APT (Debian, Ubuntu, Mint, Pop!_OS)

*** Check Availability
#+BEGIN_SRC bash
which apt
# Returns: /usr/bin/apt (if available)
#+END_SRC

*** List Installed Packages
#+BEGIN_SRC bash
apt list --installed
#+END_SRC

Output format:
#+BEGIN_EXAMPLE
vim/jammy,now 2:8.2.3995-1ubuntu2 amd64 [installed]
git/jammy,now 1:2.34.1-1ubuntu1 amd64 [installed]
#+END_EXAMPLE

Parsing: Split on '/', take first part as package name

*** Install Packages
#+BEGIN_SRC bash
sudo apt install -y vim git curl
#+END_SRC

Flags:
- -y: Auto-confirm (non-interactive)
- --no-install-recommends: Skip recommended packages

*** Remove Packages
#+BEGIN_SRC bash
sudo apt remove -y vim
sudo apt purge -y vim  # Also remove config files
#+END_SRC

*** List Orphans
#+BEGIN_SRC bash
apt autoremove --dry-run
#+END_SRC

*** Remove Orphans
#+BEGIN_SRC bash
sudo apt autoremove -y
#+END_SRC

*** Common Errors
| Error | Cause | Solution |
|-------+-------+----------|
| E: Could not get lock | Another apt process running | Wait or kill other process |
| E: Unable to locate package | Package not in repos | Check spelling, update apt |
| E: Unmet dependencies | Dependency conflict | Run apt --fix-broken install |

** Pacman (Arch, Manjaro, EndeavourOS)

*** Check Availability
#+BEGIN_SRC bash
which pacman
#+END_SRC

*** List Installed Packages
#+BEGIN_SRC bash
pacman -Q
#+END_SRC

Output format:
#+BEGIN_EXAMPLE
vim 9.0.1000-1
git 2.39.0-1
#+END_EXAMPLE

Parsing: Split on space, take first part

*** List Explicitly Installed (not dependencies)
#+BEGIN_SRC bash
pacman -Qe
#+END_SRC

*** Install Packages
#+BEGIN_SRC bash
sudo pacman -S --noconfirm vim git
#+END_SRC

Flags:
- -S: Sync (install)
- --noconfirm: Skip confirmation
- --needed: Skip if already installed

*** Remove Packages
#+BEGIN_SRC bash
sudo pacman -R --noconfirm vim      # Keep dependencies
sudo pacman -Rs --noconfirm vim     # Remove with unused deps
sudo pacman -Rns --noconfirm vim    # Also remove config files
#+END_SRC

*** List Orphans
#+BEGIN_SRC bash
pacman -Qtdq
#+END_SRC

Returns package names only, one per line. Empty if no orphans.

*** Remove Orphans
#+BEGIN_SRC bash
sudo pacman -Rns $(pacman -Qtdq)
#+END_SRC

*** Common Errors
| Error | Cause | Solution |
|-------+-------+----------|
| unable to lock database | Another pacman running | Wait or remove /var/lib/pacman/db.lck |
| target not found | Package doesn't exist | Check spelling, sync database |
| conflicting files | File already exists | Use --overwrite or resolve manually |

* GitHub Releases

** Check Availability
Always available (uses HTTP API)

** List "Installed" Packages
Read from tracking file:
#+BEGIN_SRC bash
cat ~/.config/nexus/githubInstalled.txt
#+END_SRC

Format:
#+BEGIN_EXAMPLE
lazygit=0.40.2
delta=0.16.5
#+END_EXAMPLE

** Install Process
1. Query GitHub API for latest release
2. Find matching asset for architecture
3. Download asset
4. Install based on type:
   - .deb: sudo dpkg -i file.deb
   - .rpm: sudo rpm -i file.rpm
   - .tar.gz: Extract, chmod +x, copy to path
   - .AppImage: chmod +x, copy to path
   - .zip: Extract, chmod +x, copy to path

** API Endpoint
#+BEGIN_SRC bash
# Latest release
curl -H "User-Agent: nexus" \
  https://api.github.com/repos/OWNER/REPO/releases/latest

# With auth (higher rate limit)
curl -H "User-Agent: nexus" \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  https://api.github.com/repos/OWNER/REPO/releases/latest
#+END_SRC

** Asset Selection Logic
#+BEGIN_EXAMPLE
1. Get system architecture: x86_64, aarch64, arm
2. Get system OS: linux, darwin, windows
3. Filter assets by:
   - Contains architecture string (x86_64, amd64, 64)
   - Contains OS string (linux, Linux)
   - Matches requested type (.deb, .tar.gz, etc.)
4. Select best match
#+END_EXAMPLE

** Remove Process
1. Delete installed file
2. Remove entry from githubInstalled.txt

* Output Parsing Patterns

** Rust Code Patterns

*** APT Parsing
#+BEGIN_SRC rust
fn parse_apt_list(output: &str) -> Vec<String> {
    output
        .lines()
        .filter(|line| !line.is_empty() && !line.starts_with("Listing"))
        .filter_map(|line| line.split('/').next())
        .map(String::from)
        .collect()
}
#+END_SRC

*** Pacman Parsing
#+BEGIN_SRC rust
fn parse_pacman_q(output: &str) -> Vec<String> {
    output
        .lines()
        .filter(|line| !line.is_empty())
        .filter_map(|line| line.split_whitespace().next())
        .map(String::from)
        .collect()
}
#+END_SRC

*** Pip Parsing
#+BEGIN_SRC rust
fn parse_pip_freeze(output: &str) -> Vec<String> {
    output
        .lines()
        .filter(|line| !line.is_empty() && !line.starts_with('#'))
        .filter_map(|line| line.split("==").next())
        .map(String::from)
        .collect()
}
#+END_SRC

*** Cargo Parsing
#+BEGIN_SRC rust
fn parse_cargo_list(output: &str) -> Vec<String> {
    output
        .lines()
        .filter(|line| !line.starts_with(' ') && !line.is_empty())
        .filter_map(|line| {
            // "ripgrep v13.0.0:" -> "ripgrep"
            line.split_whitespace().next()
        })
        .map(String::from)
        .collect()
}
#+END_SRC

*** NPM Global Parsing
#+BEGIN_SRC rust
fn parse_npm_global(output: &str) -> Vec<String> {
    output
        .lines()
        .filter(|line| line.contains("├──") || line.contains("└──"))
        .filter_map(|line| {
            // "├── typescript@4.9.5" -> "typescript"
            line.split(&['├', '└', '─', ' '][..])
                .last()
                .and_then(|s| s.split('@').next())
                .map(|s| s.trim().to_string())
        })
        .filter(|s| !s.is_empty())
        .collect()
}
#+END_SRC

* Quick Copy-Paste Commands

** Test All Backends Availability
#+BEGIN_SRC bash
echo "=== Checking backend availability ==="
for cmd in apt pacman dnf zypper apk flatpak snap brew pip pipx npm yarn bun cargo gem composer go; do
    if which $cmd > /dev/null 2>&1; then
        echo "✓ $cmd: $(which $cmd)"
    else
        echo "✗ $cmd: not found"
    fi
done
#+END_SRC
